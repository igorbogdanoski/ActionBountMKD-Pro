## 🔧 Технички чекори пред подача

### Чекор 1 — Ажурирај Firebase за нов package name

Откако ќе регистрираш Google Play Developer акаунт:

1. Одди на [Firebase Console](https://console.firebase.google.com) → Project `avanturakreator`
2. Project Settings → Your apps → Add Android app
3. Package name: `net.mismath.avantura`
4. Преземи го новиот `google-services.json`
5. Стави го во `apps/mobile/google-services.json` (или EAS secret)
6. Генерирај SHA-1 fingerprint:
   ```bash
   eas credentials --platform android
   ```
7. Додади SHA-1 во Firebase → Android app → SHA certificate fingerprints

### Чекор 2 — EAS Build (Production AAB)

```bash
cd apps/mobile
eas build --platform android --profile production
```

Ова генерира `.aab` (Android App Bundle) — го поставуваш на Play Console.

### Чекор 3 — Регистрирај Google Play Developer акаунт

1. https://play.google.com/console/signup
2. Плати $25 (one-time, со PayPal или картичка)
3. Верификација трае 2-3 работни дена

### Чекор 4 — Создај апликација во Play Console

1. Create app → App name: „Авантура — GPS Квестови"
2. Default language: Macedonian (mk)
3. App or game: App
4. Free or paid: Free (монетизација преку in-app)
5. Пополни Store Listing (текстот горе)
6. Upload screenshots и feature graphic
7. Upload AAB (од Чекор 2)
8. Пополни Content Rating
9. Пополни Data Safety
10. Publish → Internal testing → потоа Production

### Чекор 5 — TWA (Trusted Web Activity) — Алтернатива

Ако сакаш побрза патека без Expo build:
- TWA обвивка около `avantura.mismath.net` (PWA)
- Пакет: `net.mismath.avantura`
- Бара: `assetlinks.json` на `/.well-known/assetlinks.json` на доменот
- Алат: [Bubblewrap CLI](https://github.com/GoogleChromeLabs/bubblewrap)
